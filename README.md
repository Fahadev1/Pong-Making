# Pong

A classic Pong clone built in Godot 4 my first game project, made to learn Godot's scene system, signals, and basic 2D physics.


## How to play
- **Player 1:** W / S keys
- **Player 2:** Arrow Up / Arrow Down


## What I built
- Ball physics with collision response and angle variation based on paddle hit position
- Score tracking and HUD
- Independent paddle controls
- Reset/serve logic after each point

## Built with
- Godot 4
- GDScript
